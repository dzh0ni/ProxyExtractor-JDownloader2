Extractor de Proxies para JDownloader 2

Descripción

Extractor de Proxies para JDownloader 2 es una herramienta en PowerShell y Bash para automatizar la extracción y conversión de listas de proxies, haciendo que sean compatibles con JDownloader 2. Esto permite descargar más rápido y con mayor anonimato

Instalación

  1. Descarga el archivo ZIP del repositorio: Descargar el repositorio
  2. Extrae el contenido del archivo ZIP en una carpeta de tu elección.
  3. En Linux: Ejecuta los comandos desde la terminal: 
    sudo rm -rf ProxyExtractor-JDownloader2
sudo git clone https://github.com/dzh0ni/ProxyExtractor-JDownloader2.git
sudo chmod +x ProxyExtractor-JDownloader2/Linux/*
cd ProxyExtractor-JDownloader2/Linux
ls -ltha

Modo de Uso

  1. Navega al directorio del script.
  2. Ejecuta el script:
    • Windows: RunGetProxies_JDownloader.cmd
    • Linux: ./Proxies_JDownloader.sh
  3. Visualiza los resultados:
    • Progreso en consola.
    • Archivos generados:
      • proxies.txt
      • proxies_jdownloader.txt

Ejemplo de Salida

proxies.txt

46.46.56.64:1080
195.98.77.179:1080
192.111.134.10:4145

proxies_jdownloader.txt

socks5://46.46.56.64:1080
socks5://195.98.77.179:1080
socks5://192.111.134.10:4145

Notas

  • Ejecuta con permisos adecuados según el sistema operativo.
  • Guarda los archivos generados para mantenerlos organizados.
  • Elimina archivos no necesarios para evitar accesos no autorizados.

Características

  • Automatización: Procesa y convierte proxies automáticamente.
  • Organización: Archivos separados para fácil consulta.
  • Multiplataforma: Compatible con Windows y Linux.
  • Actualización: Funciones para mantener el script actualizado.

Requisitos

  • Sistema Operativo: Windows o Linux
  • Dependencias: PowerShell (Windows) o Bash (Linux)
